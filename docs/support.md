# ColaMeta Support

ColaMeta support covers the public ChatGPT plugin listing, the ColaMeta MCP
connector, OAuth sign-in, tool discovery, tool behavior, privacy questions, and
requests concerning project artifacts created by ColaMeta.

## Request support

Use the public repository and contact its owner:

- Repository: https://github.com/JENN2046/colameta
- Maintainer profile: https://github.com/JENN2046

When requesting help, include the affected workflow, the approximate time of
the problem, the tool name, and a redacted error summary. Never include bearer
tokens, cookies, OAuth authorization codes, client secrets, private keys, raw
provider responses, or private project contents.

## Security and privacy reports

Do not publish credentials or sensitive project data in a public report.
Contact the repository owner through the maintainer profile above and provide
only the minimum information needed to establish a private follow-up channel.

## Service expectations

ColaMeta is provided on a best-effort basis. Response times and service
availability are not guaranteed. Review or publication through the OpenAI
plugin directory does not create a separate service-level agreement.
