## 目标

为 v1.6 的 runtime version status decision contract 增加一个纯函数 evaluator。

`Evaluator` = 评估器。中文意思是：输入 `restart_needed_state`，输出结构化决策
对象；它只判断状态，不执行任何副作用。

## 输出对象

evaluator 必须返回结构化 decision object，包含：

- `decision_kind`
- `operator_notice_required`
- `diagnostics_required`
- `forbidden_actions`

## 必需行为

### `restart_needed_state == "not_needed"`

- `decision_kind`: `normal_operation`
- `operator_notice_required`: `false`
- `diagnostics_required`: `false`
- 不提示 restart；
- `forbidden_actions` 仍必须包含 restart、reload、kill、apply 和 mutation actions。

### `restart_needed_state == "needed"`

- `decision_kind`: `operator_handoff_notice`
- `operator_notice_required`: `true`
- `diagnostics_required`: `false`
- 输出必须说明这只是 operator handoff；
- `forbidden_actions` 必须包含 automatic restart、reload、kill、apply 和 mutation actions。

### `restart_needed_state == "unknown"`

- `decision_kind`: `bounded_read_only_diagnostics`
- `operator_notice_required`: `false`
- `diagnostics_required`: `true`
- 输出必须说明 unknown 不安全，也不授权动作；
- `forbidden_actions` 必须包含 automatic restart、reload、kill、apply 和 mutation actions。

### invalid、missing 或 unexpected state

- fail closed 为 `bounded_read_only_diagnostics`；
- `diagnostics_required`: `true`；
- 不授权任何 automatic action。

## 实现要求

- 新增 `runner/runtime_version_decision.py`。
- 新增 `tests/test_runtime_version_decision.py`。
- evaluator 必须是 pure function：无 file IO、无 subprocess、无 shell、无 network、无
  state mutation、无 Git operation、无 service lifecycle operation。
- 不修改 `runner/runtime_observability.py`。
- 不修改 `runner/mcp_server.py`。
- 不注册新 MCP tool。
- 不增加 Web route。
- 不增加 restart、reload、kill、apply 行为。

建议函数名：`evaluate_runtime_version_status_decision`

建议常量名：`FORBIDDEN_RUNTIME_VERSION_STATUS_ACTIONS`

`forbidden_actions` 至少包括：

- restart
- reload
- kill
- apply
- service_lifecycle_mutation
- state_mutation
- config_write
- git_fetch
- git_pull
- git_push
- git_tag
- git_release
- executor_workflow_mutation
- web_route_mutation

## 测试要求

测试必须验证：

- not_needed 返回 normal_operation，不需要 operator notice，不需要 diagnostics；
- needed 返回 operator_handoff_notice，且 `operator_notice_required` 为 true；
- unknown 返回 bounded_read_only_diagnostics，且 `diagnostics_required` 为 true；
- invalid input fail closed 到 bounded_read_only_diagnostics；
- 每个 decision 都包含 `forbidden_actions`；
- `forbidden_actions` 包含 restart、reload、kill、apply 和 mutation actions；
- evaluator 无副作用，且不暴露可执行 restart、reload、kill、apply callable。
